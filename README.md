# ✦ Bible Companion

A comprehensive, offline-first Bible study reference — all 66 books, five interactive tabs, zero dependencies, one file.

## What's Inside

| Tab | Content |
|-----|---------|
| **Old Testament** | Reading notes, themes, and context for all 39 OT books |
| **New Testament** | Reading notes, themes, and context for all 27 NT books |
| **Life & Customs** | Daily life, culture, and practices of the biblical world |
| **Revelation** | Chapter-by-chapter study guide with symbols, structure, and interpretive frameworks |
| **AI Prompts** | Context-aware prompt generator for 66 books × 6 study modes — paste into any AI for deep study |

## Use It

**On iPhone/iPad:**  
Open the hosted link in Safari → Share → **Add to Home Screen**  
Works offline as a standalone app — no browser chrome, no internet needed after first load.

**On Desktop:**  
Open `index.html` in any browser. That's it.

## Technical Details

- **Single file** — everything is in `index.html` (1.2 MB)
- **Zero dependencies** — no frameworks, no CDN, no external requests
- **Offline-first** — works without internet once loaded
- **PWA-ready** — manifest, icons, and meta tags embedded for Add to Home Screen
- **Secure** — Content Security Policy, sandboxed iframes, no eval, no fetch
- **Accessible** — keyboard navigation, semantic HTML, respects `prefers-reduced-motion`

## Hosting

This site is designed for [GitHub Pages](https://pages.github.com/). To deploy your own:

1. Fork this repo (or create a new repo and upload `index.html`)
2. Go to **Settings → Pages**
3. Under Source, select **main** branch, root folder
4. Click Save — your site will be live at `https://yourusername.github.io/repo-name`
